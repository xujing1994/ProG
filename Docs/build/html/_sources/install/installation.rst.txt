Installation
============

.. tip::
    We did our testing in the following **environment**
        - Python 3.9.17
        - PyTorch 2.0.1
        - torch-geometric 2.3.1

.. Warning::
    *for MAC User with M1 chip*
        - You may encounter many issues when installing and executing
        - You may use a server environment such as copolit




Installation via Anaconda
-------------------------

You can now install proG via Anaconda, simply run

.. code-block:: none

   conda install proG


Installation via PyPi
---------------------

For this, simply run:

.. code-block:: none

   pip install proG

.. note::
    Corresponding deployments are **being** prepared
