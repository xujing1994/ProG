The Projected Randomized Block Coordinate Descent (PRBCD) adversarial attack from the Robustness of Graph Neural Networks at Scale paper.

DataLoaderMasking

Data loader which merges data objects from a torch_geometric.data.dataset to a mini-batch. Args: dataset (Dataset): The dataset from which to load the data. batch_size (int, optional): How may samples per batch to load. (default: 1) shuffle (bool, optional): If set to True, the data will be reshuffled at every epoch (default: True).
